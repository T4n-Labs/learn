# Pengenalan

Python dikembangkan oleh [Guido Van Rossum](https://id.wikipedia.org/wiki/Guido_van_Rossum), Programmer asal Belanda. Python menggunakan pemograman C sebagai basisnya.

Python adalah Bahasa pemograman multiparadigma, yaitu mendukung berorientasi objek, pemograman terstruktur dan pemograman fungsional

Kelebihan Python :
- Mudah Dipelajari
- Mudah Digunakan
- Mendukung IoT(Internet Od Things)
- Multiplatform
